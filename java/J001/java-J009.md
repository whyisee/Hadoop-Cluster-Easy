## JVM,JDK,JRE

### JVM
java虚拟机,是运行java字节码的虚拟机,也就是执行.class文件的环境,是java实现"一次编译,到处运行"的关键.  
java程序从源代码到执行分为三步:  
源代码(.java文件) - 编译器 -> 字节码(.class文件) - JVM -> 机器码
